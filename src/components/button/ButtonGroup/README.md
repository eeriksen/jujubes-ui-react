# ButtonGroup

### Description
Use a `<ButtonGroup />` when you want to group two or more buttons nicely together.

### Usage
```jsx
<ButtonGroup>
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
</ButtonGroup>
```

### Properties
| Property | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `className` | string | *no* | null | Custom class name. |
