# Select

Use the `<Select />` when the user should choose from a list of alternatives.

<Props of={Select} />

### Basic
<Preview>
    <Story name="basic" >
        {() => {
            const [value, setValue] = useState(null);
            return (
                <Form>
                    <FormItem>
                        <Select placeholder="Choose one" value={value} onChange={setValue}>
                            <Option value={0}>
                                First choice
                            </Option>
                            <Option value={1}>
                                Second choice
                            </Option>
                            <Option value={2}>
                                Third choice
                            </Option>
                            <Option value={3}>
                                Fourth choice
                            </Option>
                        </Select>
                    </FormItem>
                </Form>
            )
        }}
    </Story>
</Preview>
