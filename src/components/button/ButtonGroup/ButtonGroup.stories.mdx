# ButtonGroup

Use a `<ButtonGroup />` when you want to group two or more buttons nicely together.

<ArgsTable of={ButtonGroup} />

### Basic

<Canvas>
    <Story name="basic">
        {() => {
            const [index, setIndex] = useState(0);
            return (
                <Row>
                    <Col span={24}>
                        <ButtonGroup>
                            <Button active={index === 0} onClick={() => setIndex(0)}>
                                Left
                            </Button>
                            <Button active={index === 1} onClick={() => setIndex(1)}>
                                Middle
                            </Button>
                            <Button active={index === 2} onClick={() => setIndex(2)}>
                                Right
                            </Button>
                        </ButtonGroup>
                    </Col>
                    <Col span={24}>
                        <ButtonGroup fill>
                            <Button active={index === 0} onClick={() => setIndex(0)}>
                                Left
                            </Button>
                            <Button active={index === 1} onClick={() => setIndex(1)}>
                                Middle
                            </Button>
                            <Button active={index === 2} onClick={() => setIndex(2)}>
                                Right
                            </Button>
                        </ButtonGroup>
                    </Col>
                </Row>

        }}
    </Story>
