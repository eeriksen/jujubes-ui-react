# Spinner

### Description
The `<Spinner />` component is great for showing the user that something is happening.

### Usage
```javascript
import { Spinner } from "tv2-ui-react"
```

```jsx
<Spinner />
```


### Properties
| Property | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `large` | bool | *no* | false | Show a large spinner. |
