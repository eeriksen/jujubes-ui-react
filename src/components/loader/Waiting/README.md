# Waiting

### Description
The `<Waiting />` component is great for showing the user that something is happening.

### Usage
```javascript
import { Waiting } from "@tv2no/ui-react"
```

```jsx
<Waiting />
```


### Properties
| Property | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `large` | bool | *no* | false | Show a large spinner. |
