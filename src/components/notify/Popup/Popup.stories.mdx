# Popup

Popups has many use cases; Take input from the user, alert of changes, get user confirmation and more.
The toolkit gives you multiple types of popup-components that you can use for all of these cases.
In a basic setup, the `<Popup />` component wraps the popup. `<PopupTitle />` let's you display a title inside your popup.
For any custom content in the popup, wrap it in the `<PopupContent />` component, and navigational buttons inside
the `<PopupFooter />` component.

With `<PopupTabs />` you can add tabbed content inside the popup the same way you would use
`<Tabs />` normally.

<ArgsTable of={Popup} />

# PopupTitle

The `<PopupTitle/>` component displays a title and description in the popup.
<ArgsTable of={PopupTitle} />

# PopupContent

Wrap all regular content of the popup inside the `<PopupContent/>` component.
<ArgsTable of={PopupContent} />

# PopupFooter

The `<PopupFooter />` is where you place any navigation and form buttons. Layout multiple elements with `<Row/>` and `<Col/>`.
<ArgsTable of={PopupFooter} />

# PopupTabs

If you want to use tabs inside a popup, wrap the `<Tabs/>` component inside the `<PopupTabs/>` component.
<ArgsTable of={PopupTabs} />
